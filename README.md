# Figma ComponentCraft 插件

这是一个Figma插件，允许用户上传PNG图片，填写组件名称和描述，然后自动将图片等比裁剪为300x300px并创建为组件。

## 功能特点

- 上传PNG图片
- 填写组件名称和描述
- 自动等比裁剪图片至300x300px
- 创建命名组件并添加描述
- 实时预览上传的图片

## 安装方法

1. 在Figma中打开插件管理界面
2. 点击"开发"选项卡
3. 点击"+"按钮，选择"导入插件"
4. 选择本插件的`manifest.json`文件

## 使用方法

1. 在Figma中打开或创建一个文件
2. 运行"ComponentCraft"插件
3. 点击"选择图片"按钮，上传PNG格式的图片
4. 填写组件名称和描述
5. 点击"创建组件"按钮
6. 插件将自动创建300x300px的组件，并将图片等比缩放居中显示

## 技术实现

- 使用Figma插件API创建组件和图片节点
- 实现等比缩放算法，确保图片保持原始比例
- 提供用户友好的界面，支持实时预览
- 包含完整的错误处理和用户反馈机制
